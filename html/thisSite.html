About this site

First off, thank you for coming to my site and looking around! I really appreciate it. Now, let me tell you more about this site. I made this site Bootstrap for styling, and any icons you see were from the free open source icon library from Bootstrap! Thank you Bootstrap! I did a tiny bit of CSS> to override some of the default stylings, but I tried to use the bootstrap classes as much as possible without having to write CSS. To send emails from the contact page I am using the smptjs cdn along with Elastic Email for smtp server hosting. To host the site I am using the W3Schools spaces feature which allows you to upload files to a "space" that will host the files for you and give you a custom URL. You can pay for a domain if you wish but I wanted to keep this whole project free which is nice! I don't get a lot of space to use with the free version though and if too many requests come to my space then they will not allow anymore. So please don't spam the site for requests! If you wish to contact me feel free to by using the Contact Me! page if you haven't!


What I used for this site:

  • HTML
  • CSS
  • JavaScript
  • Bootstrap and some of its free icons
  • W3Schools was used to host this site!
  • smtpjs was used to send emails from the contact page!
  • elasticemail was used to host the smtp server for the contact page!
