About this site

First off, thank you for coming to my site and looking around! I really appreciate it. Now, let me tell you more about this site. I made this site with Bootstrap for styling, and any icons you see were from the free open source icon library from Bootstrap! Thank you Bootstrap! I did a tiny bit of CSS to override some of the default stylings, but I tried to use the bootstrap classes as much as possible without having to write CSS. To send emails from the contact page I am using the smptjs cdn along with Elastic Email for smtp server hosting. To host the site I am using AWS S3 buckets for static website hosting, I also got the domain through AWS as well! If you wish to contact me feel free to by using the Contact Me! page if you haven't!


What I used for this site:

  • HTML
  • CSS
  • JavaScript
  • Bootstrap and some of its free icons
  • AWS was used to host this site!
  • smtpjs was used to send emails from the contact page!
  • elasticemail was used to host the smtp server for the contact page!
