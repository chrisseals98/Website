Linux Projects

In Fall of 2022 I took a Unix fundamentals class where we worked with linux operating systems in a command line environment to do simple tasks such as file manipulation and text manipulation. We also worked with virtual machines where we spun up instances of a linux OS to test and learn! Besides working directly in the command line, we also learned some introduction Bash scripting as well. With these scripts we did simple tasks to learn more about Bash. Some of these tasks involve taking data and converting it into JSON, and taking the list of users that logged into a server and formatting a list from that data! These scripts and more can be found from the github linked below!
